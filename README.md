# 100-Day Lean Bulk Tracker

## Overview
The **100-Day lean bulk Tracker** is a web application designed to help users track their daily nutrition intake during a lean bulking phase. The application fetches real-time nutritional data from public APIs and allows users to monitor calories, protein, carbs, and fats efficiently.

This project demonstrates API integration, JavaScript fundamentals, and interactive UI development.

---

## Objective
- Track daily food intake over a structured 100-day lean bulk plan
- Provide accurate nutritional data using external APIs
- Enable users to search, filter, and analyze food items
- Help users stay consistent with their fitness goals

---

## APIs Used

### 1. USDA FoodData Central
- Free and government-maintained API
- Provides detailed nutritional data
- Requires API key registration
- Website: https://fdc.nal.usda.gov/

### 2. Edamam Food Database
- Freemium API
- Limited requests in free tier (approx. 10–50 requests/min)
- Useful for enhanced food search and parsing
- Website: https://developer.edamam.com/

---

## Features

### Core Features
- Search food items by name
- Display nutritional values (calories, protein, fats, carbs)
- Track daily intake
- Add/remove food items from tracker

### Interactive Features (Milestone 3)
- Search functionality using array methods
- Filter foods based on nutrients (e.g., high protein)
- Sort results (e.g., calories ascending/descending)
- Responsive UI for mobile, tablet, and desktop
